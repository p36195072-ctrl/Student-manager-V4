Student Manager V4

A beginner-friendly Python project that demonstrates how to manage student records using dictionaries, validation, exception handling, and a menu-driven interface.

📌 Description

This program allows users to:

Add students
Search for students
View all students
Delete students
Track the total number of students
Exit the application

Student data is stored using nested dictionaries containing age, city, and class information.

🧠 Concepts Used

Dictionaries
Nested Dictionaries
CRUD Operations
User Input
Input Validation
while True Loop
if-elif-else
try-except
ValueError
len()
del
Dictionary Iteration

💻 Features

✅ Add Student Records

✅ Search Students

✅ View Student Information

✅ Delete Students

✅ Input Validation

✅ Exception Handling

✅ Total Student Counter

✅ Menu-Driven Program

💻 Code Highlights

Prevents empty names, ages, cities, and class fields.
Handles invalid class numbers using try-except.
Allows searching students by name.
Supports deleting existing students.
Shows total number of student records.
